# NVM

## Install

https://tecadmin.net/install-nvm-centos-8

## Listing all installed versions

```bash
nvm list
```

## Install

```bash
nvm install ${VERSION}
```

## Uninstall

```bash
nvm uninstall ${VERSION}
```

## Usage

```bash
nvm use ${VERSION}
```

## Set default version

```bash
nvm alias default ${VERSION}
```
